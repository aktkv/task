Введение
В этом документе представлена документация для приложения Записная Книжка, разработанного с использованием шаблона проектирования Adapter.

Цели
Цель этого проекта - создать приложение Записная Книжка с использованием шаблона Адаптер. Этот проект предназначен для изучения и демонстрации использования шаблона Адаптер с помощью языка программирования Java.

Архитектура и компоненты
Приложение Записная Книжка состоит из следующих компонентов:

– Адаптер - это интерфейс, который позволяет компонентам, работающим на разных уровнях абстракции, взаимодействовать друг с другом. В данном проекте адаптер используется для соединения записной книжки с  форматами данных.
– Записная книжкa - это основной компонент приложения, который представляет собой хранилище для контактной информации.
– Форматы данных - это компоненты, которые предоставляют различные способы хранения контактной информации. В этом проекте используются два формата данных: CSV и JSON.
